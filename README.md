# 犯罪雷達

使用python為開發語言，Database 為 Sql Server 2012 以上，以 Flask 為應用程式開發基礎框架。

專案目前分成5個 目錄夾：

1. core：共用函式庫。
2. model：提供bert深度學習模型。
3. static：靜態資源存放。
4. task：排程的程式。
5. templates：存放網頁模板。
